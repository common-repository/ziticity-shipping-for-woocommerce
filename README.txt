== Description ==

ZITICITY shipping extension for WooCommerce.

Features of ZITICITY plugin:

1. Shipping method availability based on recipient’s address
2. Multiple pickup locations (warehouses)
3. Multi-package deliveries
4. Recipient’s age check in case the delivery contains alcohol or energy drinks
5. Shipping labels
6. Real-time tracking - once the goods are ready for delivery, the receiver gets a text message. Then you can monitor the process and track the delivery path on the map in real time.
7. Free delivery over a certain shopping cart value
8. Weight-based delivery price

For more information, please visit ZITICITY website:
[https://ziticity.com/solutions/ziticity-for-e-commerce/](https://ziticity.com/solutions/ziticity-for-e-commerce/)

**Available services:**

1. Express, same-day and parcel locker delivery service in the cities where ZITICITY operates
2. Collection of money by card with COD service
3. Multi-package deliveries
4. Recipient’s age check in case the packages contain alcohol or energy drinks

For more information, please visit ZITICITY website:
[https://ziticity.com/solutions/ziticity-for-e-commerce/](https://ziticity.com/solutions/ziticity-for-e-commerce/)

**Prerequisites:**

* This extension is available for the markets ZITICITY operates in. You can see the list of the markets on ziticity.com website.
* In order to use the extension, you must have a ZITICITY account.

== Installation ==

There are two methods to install the plugin: using WordPress Plugin installer and manually:

Using the WordPress Plugin Installer:

1. Unzip the downloaded ZITICITY plugin .ZIP file into a new directory.
2. In your WordPress admin panel go to **Plugins > Add New > Upload Plugin**.
3. Upload the file **woocommerce-ziticity.zip** which is in the directory you created in step 1.
4. Click the **Install Now** button.
5. After the installation is completed, click the button **Activate Now**.

Manual Installation:

1. Unzip the downloaded ZITICITY plugin .ZIP file into a new directory.
2. Navigate to this directory and find the file **woocommerce-ziticity.zip**.
3. Extract **woocommerce-ziticity.zip** into a new directory.
4. Navigate to the newly extracted directory. You will notice it contains a directory called **woocommerce-ziticity**.
5. Upload the contents of the **woocommerce-ziticity** directory to **wp-content > plugins**, making sure to preserve the directory structure.
6. Go to your WordPress admin panel **Plugins > Installed Plugins > WooCommerce ZITICITY Shipping** and click **Activate**.

Congratulations! ZITICITY is now installed.
